# 📊 YouTube 인기 영상 분석 프로젝트

## 1. 프로젝트 개요
본 프로젝트는 YouTube 영상 데이터를 활용하여  
조회수에 영향을 미치는 요인을 분석하는 것을 목표로 한다.

### 🎯 연구 질문
> 유튜브 영상의 조회수는 좋아요 수, 댓글 수, 제목 길이, 업로드 시점, 태그 수, 썸네일 요소와 어떠한 관계를 가지는가?

---

## 2. 프로젝트 목적
- 유튜브 영상 성과(조회수)의 결정 요인 분석
- 데이터 기반 콘텐츠 전략 인사이트 도출
- 직관이 아닌 실제 데이터 기반 분석 수행

---

## 3. 데이터 수집

### 🔧 수집 방법
- YouTube Data API 활용
- Python 기반 크롤링

### 📂 카테고리
- 노래 / 먹방 / 뉴스 / ASMR / 브이로그

### 📊 데이터 규모
- 목표: 카테고리별 1000개
- 실제: 약 600~800개
- 총 데이터: 약 3,500개

### 📌 카테고리 선정 기준
- Google Trends 인기 키워드 분석
- 워드클라우드 기반 주요 관심 주제 도출

---

## 4. 데이터 전처리

### ✔ 수행 과정
- 결측치 처리 (0 대체 / 일부 제거)
- 데이터 타입 변환 (문자열 → 숫자)
- 날짜 데이터 변환
- 파생 변수 생성
  - 제목 길이
  - 업로드 시간/요일
  - 태그 수
  - 썸네일 인물 여부
- 중복 제거 (video_id 기준)
- 이상치 제거 (IQR 방식)

---

## 5. 탐색적 데이터 분석 (EDA)

### 📈 주요 결과
- 조회수/좋아요/댓글 → 오른쪽 치우친 분포
- 조회수 ↔ 좋아요 → 강한 양의 관계
- 조회수 ↔ 댓글 → 중간 관계
- 태그 수 → 음의 관계
- 썸네일 인물 포함 시 조회수 증가 경향
- 카테고리별 조회수 차이 존재

---

## 6. 주요 인사이트

- 조회수는 좋아요, 댓글과 밀접한 관계
- 태그 수는 성과와 직접적인 양의 관계 아님
- 썸네일(인물 포함)이 조회수에 영향
- 콘텐츠 유형(카테고리)에 따라 성과 구조 다름

---

## 7. 향후 계획

- 데이터 추가 수집 (정확도 향상)
- 카테고리별 회귀분석
- 모델 고도화 (추가 변수 생성)
- Streamlit 기반 대시보드 구축
