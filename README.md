# 📊 YouTube 인기 영상 분석 프로젝트

## 1. 프로젝트 개요
본 프로젝트는 YouTube 영상 데이터를 활용하여  
조회수에 영향을 미치는 요인을 분석하는 것을 목표로 한다.

중간 프로젝트에서는 데이터 수집, 전처리, EDA를 중심으로 분석을 진행하였고,
최종 프로젝트에서는 교수님 피드백을 반영하여 통계 검정, 변수 검증, 부분상관분석, 다중회귀분석, Random Forest 모델링, Streamlit 대시보드 구축까지 확장하였다.

### 🎯 연구 질문
> 유튜브 영상의 조회수는 좋아요 수, 댓글 수, 제목 길이, 업로드 시점, 태그 수, 썸네일 요소와 어떠한 관계를 가지는가?

---

## 2. 프로젝트 목적
- 유튜브 영상 성과(조회수)의 결정 요인 분석
- 데이터 기반 콘텐츠 전략 인사이트 도출
- 직관이 아닌 실제 데이터 기반 분석 수행

---

## 3. 데이터 수집

### 🔧 수집 방법
- YouTube Data API 활용
- Python 기반 크롤링

### 📂 카테고리
- 노래 / 먹방 / 뉴스 / ASMR / 브이로그

### 📊 데이터 규모
- 목표: 카테고리별 1000개
- 실제: 약 600~800개
- 총 데이터: 약 3,500개

### 📌 카테고리 선정 기준
- Google Trends 인기 키워드 분석
- 워드클라우드 기반 주요 관심 주제 도출

---

## 4. 데이터 전처리

### ✔ 수행 과정
- 결측치 처리 (0 대체 / 일부 제거)
- 데이터 타입 변환 (문자열 → 숫자)
- 날짜 데이터 변환
- 파생 변수 생성
  - 제목 길이
  - 업로드 시간/요일
  - 태그 수
  - 썸네일 인물 여부
- 중복 제거 (video_id 기준)
- 이상치 제거 (IQR 방식)

---

## 5. 탐색적 데이터 분석 (EDA)

### 📈 주요 결과
- 조회수/좋아요/댓글 → 오른쪽 치우친 분포
- 조회수 ↔ 좋아요 → 강한 양의 관계
- 조회수 ↔ 댓글 → 중간 관계
- 태그 수 → 음의 관계
- 썸네일 인물 포함 시 조회수 증가 경향
- 카테고리별 조회수 차이 존재

---

## 6. 변수 검증
- 피드백을 반영하여 자동 생성 변수의 신뢰성을 검증
- 얼굴 인식 검증 정확도 96.%
- 카테고리 분류 검증 정확도 92%

## 7. 통계 분석

### 얼굴 포함 여부 검정

얼굴 포함 여부에 따른 조회수 차이를 확인하기 위해 Welch's t-test와 Mann-Whitney U Test를 수행하였다.

| 검정 방법 | p-value |
|-----------|---------|
| Welch's t-test | 0.1019 |
| Mann-Whitney U Test | 0.2480 |

### 결과

얼굴 포함 여부는 조회수에 통계적으로 유의한 영향을 주지 않는 것으로 나타났다.

---

## 8. 부분상관분석

태그 수와 조회수의 관계가 다른 변수의 영향 때문인지 확인하기 위해 부분상관분석을 수행하였다.

### 통제 변수

- 구독자 수
- 업로드 경과 시간

### 결과

| 분석 방법 | 상관계수 |
|-----------|----------|
| 단순상관 | -0.066 |
| 둘 다 통제 | -0.197 |

### 결론

통제 이후에도 태그 수와 조회수의 음의 관계는 유지되었다.

---

## 9. 다중회귀분석

조회수에 영향을 미치는 주요 변수를 확인하기 위해 수행하였다.

### 주요 결과

양(+)의 영향

- 좋아요 수
- 댓글 수
- 구독자 수
- 업로드 경과 시간

음(-)의 영향

- 태그 수
- 숫자 포함 여부
- 감성어 포함 여부

유의하지 않음

- 얼굴 포함 여부

---

## 10. Random Forest 분석

조회수 예측에 중요한 변수를 확인하기 위해 Random Forest 모델을 구축하였다.

### 모델 성능

- R² = 0.832

### 변수 중요도 TOP 5

| 순위 | 변수 |
|------|------|
| 1 | 좋아요 수 |
| 2 | 구독자 수 |
| 3 | 댓글 수 |
| 4 | 업로드 경과 시간 |
| 5 | 태그 수 |

### 결과

좋아요 수가 조회수 예측에 가장 중요한 변수로 나타났다.

## 12. 주요 인사이트

### 영향력이 큰 변수

1. 좋아요 수
2. 구독자 수
3. 댓글 수
4. 업로드 경과 시간

### 영향력이 낮은 변수

- 태그 수
- 제목 길이
- 특수문자 사용

### 영향이 거의 없는 변수

### 최종 결론
